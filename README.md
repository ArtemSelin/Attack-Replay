# Attack Replay (v2.0.0)

Attack Replay – это универсальный инструмент командной строки для тестирования функций безопасности различных сетевых средств защиты информации (СЗИ), таких как **NGFW** (Firewall, IDS/IPS, потоковый антивирус, DPI), **NDR**, **SIEM** и других решений сетевой безопасности.

## 🚀 Принцип работы

Программа принимает на вход заранее подготовленный файл или директорию с файлами в формате `pcap` / `pcapng` с записанным вредоносным или легитимным трафиком. После этого инструмент воспроизводит этот трафик с выбранного сетевого интерфейса в режиме **stateless** (без установления TCP-соединения).

Инструмент объединяет в себе возможности и гибкость таких утилит, как: `tcpreplay`, `tcprewrite`, `tcpprep` и `tcpreplay-edit`.

---

## ✨ Основные возможности

* **Редактирование L2/L3 заголовков:** Быстрая смена MAC и IP-адресов источника и назначения как в одном файле, так и пакетно для всей директории.
* **Автоопределение параметров (`--ip`):** Автоматическое определение выходного интерфейса, локального MAC-адреса и ARP-разрешение MAC-адреса целевого СЗИ по его IP-адресу.
* **Гибкое управление скоростью:** Воспроизведение на максимальной скорости (`--topspeed`), с ограничением пакетов в секунду (`--pps`) или пропускной способности (`--mbps`).
* **Различные режимы воспроизведения:** Однократный запуск, заданное количество итераций (`-c`), бесконечный цикл (`-l`) или работа строго по таймеру (`-t`).
* **Безопасность данных:** Возможность перезаписи оригиналов или генерация уникальных копий измененных pcap-файлов (`-k`).
* **Интерактивный и тихий режимы:** Пошаговая отправка пакетов по нажатию Enter (`--oneatatime`) или скрытие лишнего вывода в консоль (`-q`).

---

## 🛠 Требования и автоматическая установка

Инструмент работает в Linux-окружении (Debian, Ubuntu, Kali Linux и др.). Для работы требуются системные утилиты:
* `tcpreplay` / `tcprewrite` / `tcpprep`
* `file`

### Быстрый старт
Скрипт содержит встроенный механизм проверки зависимостей. При первом запуске он проверит наличие утилит и предложит установить их автоматически через ваш пакетный менеджер (`apt`, `yum`, `dnf`, `pacman`, `zypper`):

```bash
git clone [https://github.com/ВАШ_ПРОФИЛЬ/attack-replay.git](https://github.com/ВАШ_ПРОФИЛЬ/attack-replay.git)
cd attack-replay
chmod +x attack_replay.sh
sudo ./attack_replay.sh --help
